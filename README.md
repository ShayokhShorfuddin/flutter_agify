# flutter_agify

A Flutter package that grabs data from the Agify.io api that can predict your age using your name!

## Usage

To use this package :

* add the dependency to your `pubspec.yaml` file.

```yaml
  dependencies:
    flutter:
      sdk: flutter
    facts:
```
